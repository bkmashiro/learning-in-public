# 公开学习 📖

> 通过构建、写作和发布来学习——公开地。

这是一个聚合所有开源学习项目的 hub。每个仓库选一个主题深入研究：一篇论文、一个系统、一个可视化，一次一个。

---

## 理念

- **构建来理解** — 学习一个系统最好的方法是自己动手造一个残缺版本
- **小步发布** — 每周一个概念，而不是每年一个宏大项目
- **写下来** — 每个项目都有 README 解释核心思路
