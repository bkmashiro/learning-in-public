## 理念

大多数学习都发生在私下——笔记在文件夹里腐烂，灵感转瞬即逝，半成品项目积灰。**公开学习**反其道而行之：每篇读过的论文变成公开的分析，每个理解的系统变成一个极简实现，每个掌握的算法变成一个可视化。

这形成了一个良性循环：

- **自律** — 公开的连续记录更难中断。
- **分享** — 你的探索对他人也有价值。
- **复利** — 每一块知识相互连接，积累的价值随时间指数增长。

「学习即输出」——只有能向外表达的知识，才是真正内化的知识。这个仓库就是连接所有公开学习项目的枢纽。
